PREREQUISTE

PHP 5.0 or above

INSTALLATION

Rename inc/conf.php.in into inc/conf.php and fill in the values using the following indications: 
* FLUNKER_BASE_URL: full URL to the flunker,
* FLUNKER_CRYPT_KEY: insert a quite long character string of your choice. Do not ever change it.

Upload your files to your web space using an FTP client.
